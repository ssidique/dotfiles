# vim-workflow

> Custom vim workflows (leader key is comma).
> Based on .vimrc configuration with Vundle plugins.

- Toggle NERDTree file explorer:

`,f`

- Find current file in NERDTree:

`,v`

- Open CtrlP buffer list:

`,bb`

- Open CtrlP mixed mode (files + buffers + MRU):

`,bm`

- Open CtrlP most recently used files:

`,bs`

- Run command in tmux pane (Vimux):

`,vp`

- Run last tmux command again (Vimux):

`,vl`

- Inspect tmux runner pane (Vimux):

`,vi`

- Zoom tmux runner pane (Vimux):

`,vz`

- Comment/uncomment lines (vim-commentary):

`gcc`

- Comment/uncomment selection (visual mode):

`gc`

- Navigate to next buffer:

`Ctrl-l`

- Navigate to previous buffer:

`Ctrl-h`

- Toggle paste mode (prevents auto-indent):

`F3`

- Search in project with fugitive:

`:Ggrep {{pattern}}`

- Git status:

`:Git status`

- Git commit:

`:Git commit`

- Git diff:

`:Git diff`

- Show git blame:

`:Git blame`

- Install/update Vundle plugins:

`:PluginInstall`

- Trigger multiple cursors on word:

`Ctrl-n (press multiple times)`

- Easy motion jump to word:

`,,w`

- Easy motion jump to line:

`,,j`
