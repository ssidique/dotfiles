# fzf-commands

> Custom fzf commands and workflows configured in .zshrc.
> These are custom functions available in your shell.

- Search and open file in nvim with preview (custom: fv):

`fv`

- Search and open file in vim with preview (custom: fvim):

`fvim`

- Fuzzy cd into any directory (custom: fcd or fzd):

`fcd`

- Search file contents with ripgrep, then open at line (custom: frg):

`frg {{search-term}}`

- Search command history (custom: fh):

`fh`

- Fuzzy kill processes (custom: fkill):

`fkill`

- Fuzzy checkout git branch (custom: fbr):

`fbr`

- Browse git commits interactively (custom: fshow):

`fshow`

- Browse and manage git stashes (custom: fstash):

`fstash`

- Quick edit dotfiles (custom: fenv):

`fenv`

- Switch tmux sessions (custom: ftmux):

`ftmux`

- Built-in: Fuzzy file search (paste to command line):

`Ctrl-T`

- Built-in: Fuzzy command history search:

`Ctrl-R`

- Built-in: Fuzzy cd into directory:

`Alt-C`

- Git file search (from fzf-git.sh):

`Ctrl-G Ctrl-F`

- Git branch search (from fzf-git.sh):

`Ctrl-G Ctrl-B`

- Git commit hash search (from fzf-git.sh):

`Ctrl-G Ctrl-H`
